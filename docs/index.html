Overview

SportsSync returns full match context in one API call. Use it for previews, betting research, and content production without stitching feeds by hand.

Response shape stays fixed: all top-level sections are always present. If a section has no feed, it returns { "available": false }.

Base URL

https://hylnixnuabtnmjcdnujm.supabase.co/functions/v1

Required Sections

SectionPurpose
matchIdentity, kickoff, venue, status
teamsHome and away context
marketOdds, lines, consensus
formRecent performance context
h2hHead-to-head history
injuriesInjury reports and impact
playersKey player and prop context
trendsSituational trend context
live_stateLive match state
metadataFreshness and timestamps

Authentication

Send your API key in the x-api-key header on every request.

x-api-key: sk_live_your_key_here

Examples

JavaScript
const response = await fetch(
  "https://hylnixnuabtnmjcdnujm.supabase.co/functions/v1/match-context?match_id=b42fe447-b2b1-485f-ae6d-1559ee2b57c7",
  { headers: { "x-api-key": "sk_test_sportsync_beta_001" } }
);

const payload = await response.json();
console.log(payload.match, payload.market, payload.metadata);
Python
