Overview

SportsSync returns full match context in one API call. Great for previews, research, and content production without stitching feeds by hand.

Response shape stays fixed: all top-level sections are always present. If a source is missing, the section returns { "available": false }.

https://hylnixnuabtnmjcdnujm.supabase.co/functions/v1

Single-call match intelligence
Availability

Examples

JavaScript 
const response = await fetch(
  "https://hylnixnuabtnmjcdnujm.supabase.co/functions/v1/match-context?match_id=b42fe447-b2b1-485f-ae6d-1559ee2b57c7",
  { headers: { "x-api-key": "sk_test_sportsync_beta_001" } }
);

const payload = await response.json();
console.log(payload.match, payload.market, payload.metadata);
