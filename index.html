--- src/legacy_pipeline.py
  • Fetch raw schedules, odds, and injuries from 3 separate APIs

  • Write custom CRON jobs to calculate rest days and fatigue
  • Implement complex fallbacks when providers drop schema fields
  • Track freshness timestamps per individual data provider

  • Single request to /v1/matches/{id} returns complete state
  • Global entity resolution automatically maps standard IDs
  • Server-side computation of situational metrics and ATS trends
  • Strict OpenAPI schema validation guarantees shape consistency
  • Per-object freshness timestamps embedded natively
